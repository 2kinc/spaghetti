spaghetti.js

a framework for strings.

spaghetti.js provides helpful (or maybe just cool) functions such as:

  • Flexibly reversing strings
  • Encrypting/decrypting strings
  • Locking and hiding string values
  • Rearranging strings
  • And more!

Getting started

To use spaghetti.js, just paste this into your page:

<script src="https://2kinc.github.io/spaghetti-js/main.js"></script>

or download the script.

Docs

  • $s(string)

    The main function you use to make spaghetti strings.


  • .val

    Returns the value of the string.


  • .len

    Returns the length of the string.


  • .reverse(splitter = '')

    Reverse all characters of the string against a splitter.


  • .reversed

    Returns the reversed string.

spaghetti.js

a framework for strings.

spaghetti.js uses 1 global object spaghetti. spaghetti.js provides helpful functions such as:

  • Flexibly reversing strings
  • Encrypting/decrypting strings
  • Locking and hiding string values
  • Rearranging strings
  • And more!

Get started

To use spaghetti.js, just

paste this into your page: <script src="https://2kinc.github.io/spaghetti-js/main.js"></script>

or download the script.

Here are the functions:

  • spaghetti.reverse(text, splitter)

    This function is pretty self explanatory. spaghetti.reverse(text, splitter) takes a string and reverses it. You can leave the second parameter blank to reverse the string, or you can give a value such as " " to split on that value and preserve the split values. Example:

    spaghetti.reverse("Spaghetti and meatballs for you!")
    > "!uoy rof sllabtaem dna ittehgapS"
    spaghetti.reverse("Did you touch my spaghetti?", " ")
    > "spaghetti? my touch you Did"
